## Overview

A personal portfolio website built with vanilla HTML, CSS, and JavaScript. No frameworks, no build tools — just clean, performant code that runs anywhere.

The design language is **techy and bold** — dark neutral palette, monospace typography, grid-based layouts, and smooth reveal animations — built to let the work speak for itself.

---

## Features

- Animated hero with CSS grid background and staggered entrance
- Scroll-progress indicator
- Reveal-on-scroll animations via Intersection Observer
- Interactive project cards with click-to-open case study modals
- Full project galleries with challenge briefs and key deliverables
- Résumé section covering experience, skills, and education
- Keyboard & click-outside modal dismissal
- Fully responsive across mobile, tablet, and desktop
- Zero dependencies, zero build step

## Running Locally

No setup required. Just open the file in your browser:

```bash
# Clone the repo
git clone https://github.com/YOUR_USERNAME/portfolio.git

# Open in browser
open portfolio/index.html
```

Or use a simple local server for best results:

```bash
cd portfolio
npx serve .
# Visit http://localhost:3000
```

---

## Deployment

This site is hosted via **GitHub Pages**.

To deploy your own fork:
1. Go to **Settings → Pages**
2. Set source to `main` branch, `/ (root)` folder
3. Save — your site goes live at `https://YOUR_USERNAME.github.io/portfolio`

## Tech Stack

- **HTML5** — semantic, accessible markup
- **CSS3** — custom properties, grid, flexbox, keyframe animations
- **Vanilla JS** — Intersection Observer, modal logic, scroll tracking
- **Fonts** — [Syne](https://fonts.google.com/specimen/Syne) (display) + [Space Mono](https://fonts.google.com/specimen/Space+Mono) (monospace) via Google Fonts
- **Images** — `.avif` format for optimal compression
